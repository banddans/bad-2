# Bad 2

Håll koll på alla badtemperaturer i linköping, frontend för [linköping öppna data](https://www.linkoping.se/open/details?esc_entry=64&esc_context) fast med snabbare kartladdning, modernare UI och med fler features (historik, varning för trasiga termometrar etc.) än [deras officiella frontend](https://kartor.linkoping.se/portal/apps/sites/#/kartportalen/apps/4e274a1f91c1421ea952a34c5c7a3ee0/explore)

## Tech Stack

- Frontend: Vite + React
- UI: shadcn
- Karta: react-leaflet + OpenStreetMap + CARTO
- Backend (BaaS): Appwrite
- Analytics: Umami
